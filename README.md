# Girex POS

Backend for Girex POS App

## Features

- Users Management
- Products Management
- Categories Management
- Outlets Management  (Coming Soon)

## Installation

```bash
npm install
```

```bash
composer install
```

```bash
npm run dev
```

```bash
php artisan serve
```
